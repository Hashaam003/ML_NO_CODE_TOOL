ML No-Code Tool (Streamlit-Based Project)

The ML No-Code Tool is a web-based application developed using Streamlit with the goal of making machine learning accessible to users without requiring programming knowledge.

This project was designed to encourage users to focus on critical thinking, data understanding, and decision-making, rather than spending time on coding complexities.

Key Features:
Pre-built Machine Learning Models
Includes basic models such as regression, allowing users to train and test models easily.
Data Visualization Section
Users can upload datasets and generate graphs to better understand patterns and trends.
User-Friendly Interface
Simple and interactive UI so even beginners can work with machine learning concepts.
CSV File Upload
Users can upload their own datasets and directly apply analysis and models.
Objective:

The main objective of this project is to simplify machine learning workflows and help users:

Understand data more effectively
Make informed decisions
Learn ML concepts without coding barriers
Technology Used:
Python
Streamlit
Basic Machine Learning Libraries (e.g., NumPy, Pandas, Scikit-learn)
